◆ applyFilterGpu()

cv::Mat applyFilterGpu ( const cv::Mat & input,
FilterType type,
const FilterParams & params,
bool verbose = true )

Applies the selected filter to a grayscale image using custom CUDA kernels on the GPU. This function allocates device memory, copies the input image to the GPU, launches the appropriate CUDA kernel (such as Gaussian blur, erosion, dilation, opening, or closing), and copies the result back to the host. It can print kernel execution time if verbose is enabled.

Parameters
inputGrayscale input image (CV_8UC1).
typeFilterType enum specifying which filter to apply.
paramsFilterParams struct containing filter-specific parameters.
verboseIf true, prints kernel execution time and error messages.
Returns
Output image after filtering, or a clone of the input on error.

Applies the selected filter using custom CUDA kernels for a specified number of iterations. The filter is applied repeatedly on the GPU, and the average execution time can be measured and returned. Useful for benchmarking or repeated processing.

Parameters

verboseIf true, prints timing information for each iteration and the average.
avgMsPointer to store average execution time in milliseconds (can be nullptr).
Returns
Output image after the last iteration.
