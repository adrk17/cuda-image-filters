void benchmarkEveryKernel ( FilterParams & params,
cv::Mat image,
int iterations,
int verbose = 0 )

Benchmarks all supported filters by running each for a specified number of iterations.

For each filter type, applies the filter using OpenCV CPU, custom CUDA, and OpenCV CUDA implementations. Measures and prints the average execution time for each method. Optionally displays and saves output images.

Parameters
paramsFilter parameters to use for all filters.
imageInput image to filter.
iterationsNumber of times to run each filter for timing.
verboseVerbosity level (0 = silent, 1 = log info, 2 = log info and show images).

Compares two images and reports their differences.

Computes absolute difference, min/max/mean/median difference, and counts pixels with significant differences. Separates border and inner differences, and optionally displays the difference image and prints a histogram.

Parameters
img1First image to compare.
img2Second image to compare.
verboseVerbosity level (0 = summary, 1 = detailed, 2 = show difference image).
borderSizeSize of the border region to consider for border differences.
diffThresholdPixel difference threshold to count as significant.

Displays an image in a window, resizing it if it exceeds the specified maximum dimensions.

If the image is larger than maxWidth or maxHeight, it is scaled down to fit within those bounds. The image is then shown in a window with the given name.
